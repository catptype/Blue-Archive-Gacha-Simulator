Welcome, Sensei!

This is an unofficial web application, serving as a Gacha Simulator for the mobile game Blue Archive, developed by Nexon Games and published by Yostar (Japan) and Nexon (Global).

Have fun testing your luck with gacha draws without spending Pyroxene.

Features

  • Gacha simulator with pick-up rates similar to the mobile game
  • User account for tracking information such as collections and achievements
  • Guest mode allows you to have fun testing your luck
  • Full student list in the current game version
  • Open source and no collection of user information, because this is a case study for full-stack web development
